Principais bibliotecas nativas do Python para iniciantes

Introdução

Python é uma linguagem de programação poderosa, com diversos recursos para os programadores economizarem tempo apenas utilizando as bibliotecas que a própria linguagem oferece, além de outras construídas pela comunidade que usa a linguagem.

Bibliotecas são um conjunto de ferramentas prontas para serem usadas em variadas situações, evitando a necessidade de se construir códigos do zero em toda situação.

O python na sua versão 3.x já possui, por padrão, mais de 250 bibliotecas nativas (segundo a documentação encontrada em Python 3.9.1 documentation) para atender as mais diversas necessidades dos programadores.
