Reproduzindo audio

Pelo jeito mais simples

Para reproduzir audio em HTML5 podemos usar a tag <audio> com os parâmetros src para selecionar o arquivo e controls para que o usuário aperte o play, como mostrado a seguir

Você também pode trocar o parâmetro controls pelo parâmetro autoplay para tocar o audio sem precisar apertar o play, mas isso pode não funcionar em alguns navegadores.

Pelo jeito mais sofisticado

Se estiver tendo problemas com o formato do audio, é possível configurar para que o navegador tente reproduzir vários formatos do mesmo audio, e, se não for possível, disponibilize para o usuário a opção de download.

Para isso usamos, de novo, a tag <audio>, mas sem o parâmetro scr, e sim, com os parâmetros preload configurado para metadata (para o site carregar todo o audio apenas quando o usuário apertar o play, deixa menos pesado) e o controls (ou autoplay, como preferir...), com as tags <source> e seus parâmetros src (para selecionar o arquivo) e type (para selecionar o tipo de arquivo) para cada formato de audio. O HTML5 suporta os formatos MP3, OGG (formato open source) e WAV (muito pesado, não recomendável). Novamente: A ordem das tags <source> IMPORTA!
